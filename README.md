🏦 Banking Customer Data Analysis

📌 Project Overview
This project analyzes banking customer data to identify patterns in deposits, loans, and customer segmentation. The objective was to extract meaningful insights that help in understanding customer behavior and financial distribution.

🛠 Tools Used
SQL
Python (Pandas, Matplotlib)
MS Excel
Power BI

📊 Dataset Description
The dataset contains customer information including:
Account type
Deposit amount
Loan amount
Customer demographics
Transaction details

🔎 Analysis Performed
Data cleaning and preprocessing using Python
SQL queries to analyze loan and deposit distribution
Customer segmentation based on account balance
KPI analysis using Excel Pivot Tables
Interactive dashboard creation in Power BI

📈 Key Insights
Identified high-value customers contributing major deposit share
Observed loan concentration among specific customer segments
Analyzed deposit vs loan ratio trends
Discovered patterns in customer activity levels

📌 Business Impact
The analysis provides insights into customer segmentation and financial performance metrics, supporting data-driven decision-making.
